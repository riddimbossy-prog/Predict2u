# Predict2U Full Board Compact Cards v231

The compact mobile match design from Today’s Board now also applies to Full Board matches.

## Mobile behavior
- Shows league, kickoff, status, teams, prediction and odds first
- Hides engine votes, conflict data, coverage and action buttons initially
- `Details & actions` expands the hidden information
- Top Consensus Banker cards use the same compact pattern
- Desktop and tablet layouts retain the full information view
