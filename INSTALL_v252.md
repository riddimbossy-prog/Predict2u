# Install Predict2U v252

1. Extract the patch.
2. Copy `board.html`, `sw.js`, and `BUILD_VERSION.txt` into the repository root.
3. Replace the existing files when prompted.
4. Commit and push with: `Install v252 win-only carousel`.
5. Open the Picks page once in a private window. For the installed PWA, close and reopen it so service-worker v252 activates.

The patch does not contain `data.js`, `fixtures.js`, `track-log.json`, caches, or reports.
