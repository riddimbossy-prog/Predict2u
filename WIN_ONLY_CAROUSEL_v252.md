# Predict2U v252 — Win-Only Picks Carousel

## Change

The Picks page now shows a dedicated **Recent Wins** carousel directly below the page header.

- Displays winning consensus picks only.
- Uses verified results from the most recent seven days.
- Hides automatically when no verified wins are available.
- Scrolls at a medium, readable speed.
- Pauses on hover and respects reduced-motion settings.
- Repeats short result sets so the loop remains continuous.
- Updates the service-worker cache to v252 so the change reaches the installed PWA.
