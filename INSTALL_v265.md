# Predict2U v265 — Mobile Responsiveness + Teams Homepage

## Install
1. Extract the patch ZIP.
2. Copy every file into the repository root and replace existing files.
3. Commit and push to `main`.
4. Fully close and reopen the installed PWA so cache `v265` activates.

Suggested commit:

```text
Install Predict2U v265 mobile responsiveness and Teams homepage
```

No football-data workflow is required for this UI-only release.
