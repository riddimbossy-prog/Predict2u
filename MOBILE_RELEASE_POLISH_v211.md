# Predict2U v211 — Mobile Release Polish

## Fixed
- Engine activity avatars no longer collide with “publishing today” in the Overview metric.
- Match-card actions use a stable 2 × 2 phone grid: Details, Proof, + Slip and Share.
- Button labels stay on one line and retain a minimum 46px touch target.
- Market text and odds use separate grid columns.
- Today’s Results heading stacks cleanly on phones.
- Very narrow Fold cover screens switch the Overview metrics to one column.

## Required release rule
Every future Predict2U build must be checked at 320, 360, 393, 768, 884 and 1180 CSS pixels before delivery.
