# Predict2U Mobile-Only Board Redesign v229

The squeezed five-button mobile toolbar was removed from phones and replaced with a dedicated mobile interface.

## New mobile design
- Three compact board-view cards: All, Value and Safe
- Counts shown inside each card
- One full-width Filter Matches control
- League and search moved into a bottom-sheet filter panel
- Reset and Show Matches actions
- Desktop/tablet board toolbar remains unchanged

## Additional mobile fixes
- Cleaner board title and status layout
- No shrinking text inside controls
- Horizontal date and engine rails remain swipeable
- Personalization controls remain two-column and contained
- Extra narrow-phone and landscape rules
- Service worker cache updated to `predict2u-v229`
